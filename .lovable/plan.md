
# Финальные доработки VDI Такси — 5 изменений

## 1. Админ-панель: новая вкладка "Здоровье"

Добавляется в `AdminScreen.tsx` после вкладки "Шаблоны":

- **VM-карточки** (5 штук в ряд, responsive grid): каждая отображает статус (Online/Offline с цветной точкой), занятость, uptime, кнопку "Перезагрузить" где нужно
- **VPN-секция**: статус подключения и IP
- **Статус сервисов**: список из 10 сервисов со статусами (зеленый "Активна", желтый "Cookie истекла", красный "Ошибка")

## 2. Админ-панель: "Подписки" переименовывается в "Сервисы"

В той же вкладке добавляется:
- Колонка "Действия" с иконкой карандаша и тогглом вкл/выкл
- Клик на карандаш открывает модалку редактирования (Название, Категория dropdown, URL, Логин, Пароль с показать/скрыть, стоимость, тоггл "Активен")
- Кнопка "+ Добавить сервис" сверху таблицы (та же модалка, но пустая)

## 3. Профиль: блок "Контактные данные"

В `ProfileScreen.tsx` перед блоком избранных добавляется:
- Заголовок "Контактные данные"
- Input для Telegram username (placeholder "@username", значение "@anna")
- Кнопка "Сохранить"

## 4. Модалка бронирования: дата по умолчанию

В `DashboardScreen.tsx` при открытии модалки бронирования дата по умолчанию устанавливается на 14 фев 2026 (new Date(2026, 1, 14)).

## 5. Экран онбординга (новый)

Создается `OnboardingScreen.tsx` — 3-шаговый визард:
- **Шаг 1/3**: "Привяжите Telegram" — поле @username, кнопка "Далее"
- **Шаг 2/3**: "Добавьте на рабочий стол" — инструкция + placeholder-картинка, кнопка "Далее"
- **Шаг 3/3**: "Как это работает" — серый placeholder 16:9 видео, кнопка "Начать работу"
- Степпер сверху (3 точки)

В `Index.tsx` добавляется флаг `firstLogin = true`. После логина, если `firstLogin` — показывается онбординг вместо дашборда. После завершения онбординга — переход на дашборд и сброс флага.
